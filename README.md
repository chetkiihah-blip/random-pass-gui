# Random Password Generator

**Автор:** Сергей Должанский

## Описание
Графическое приложение на Python + Tkinter для генерации случайных паролей с настраиваемыми параметрами. Поддерживает выбор типов символов, установку длины пароля, отображение результата и ведение истории паролей в JSON-файле. Проект включает валидацию ввода, модульные тесты и правильную структуру для контроля версий.

## Функционал
- Ползунок для выбора длины пароля (4–64 символа).
- Чекбоксы: буквы, цифры, спецсимволы.
- Кнопка «Сгенерировать пароль».
- Отображение результата в поле ввода.
- Таблица истории (последние 50 паролей) с параметрами генерации.
- Валидация: проверка минимальной/максимальной длины, запрет генерации без выбранных типов символов.
- Сохранение истории в `passwords_history.json`, загрузка при старте.

## Требования
- Python 3.8+
- Только стандартная библиотека (tkinter, json, random, os)

## Запуск
1. Клонируйте репозиторий или скачайте файлы.
2. В терминале перейдите в папку проекта.
3. Запустите: `python main.py`

## Примеры использования (тестовые сценарии)
- **Позитивный:** длина 12, все типы символов → генерируется пароль из 12 символов, появляется в истории.
- **Негативный:** длина 3 → выводится предупреждение о минимальной длине.
- **Граничный:** длина 4 или 64 → пароль генерируется, ошибок нет.
- **Без типов символов:** все чекбоксы сняты → предупреждение, пароль не генерируется.

## Тестирование
Тесты написаны в `test_core.py` и используют `pytest`.  
Запуск: `pytest test_core.py`
